# Checklist Command Documentation

## Command
`speckit.checklist`

## Purpose
Generates or updates checklists for different phases of feature development.

## When to Use
- During feature planning
- Before starting implementation
- For review and quality assurance

## Prerequisites
- Feature specification should exist
- Implementation plan recommended

## Execution Flow

1. **Identify Phase**: Determine which checklist is needed
2. **Load Templates**: Use appropriate checklist template
3. **Customize**: Adapt to feature specifics
4. **Generate**: Create checklist file

## Checklist Types

- **Planning**: Pre-implementation checks
- **Implementation**: Development process checks
- **Review**: Code review criteria
- **Deployment**: Release readiness

## Output

Creates checklists in `specs/###-feature-name/checklists/` directory.

## Agent-Specific Guidance

This command is agent-agnostic and can be executed by any AI assistant.
